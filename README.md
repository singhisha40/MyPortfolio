# Personal Portfolio

A modern and responsive personal portfolio website built using React and Vite.  
This project showcases my skills, projects, experience, and contact information in a clean and interactive interface.

---

## 🚀 Tech Stack

- React
- Vite
- JavaScript / JSX
- CSS 
- Git & GitHub

---

## ✨ Features

- Modern UI/UX
- Fast performance with Vite
- Reusable React components
- Projects showcase section
- Skills and experience section
- Smooth navigation and interactive layout

---

## 📂 Project Structure

```bash
src/
│
├── components/     # Reusable UI components
├── pages/          # Website pages/sections
├── App.jsx
└── main.jsx
```

Through this project, I improved my understanding of:

- React component architecture
- State management
- Responsive web design
- Performance optimization using Vite
- Clean UI structuring and reusable code practices
